## 1. 问题复述

你这次不是要先做整条 AI 后端链路，而是要**先把“穿搭日记”页面改造成“智能提醒页”的新视觉与交互形态**，并且页面结构要贴近你给的设计草图：

1. **最上面**：固定展示一张主视觉图片（机器人 + 衣橱图）。
2. **中间**：保留当前项目已有的天气提醒能力，但要改成真正的数据绑定，不再写死文案。
3. **最下面**：改成“人机对话框”形式，用户输入需求后，智能体返回搭配建议。
4. 智能体返回结果里要有一个**箭头按钮**，点击后**跳转到试穿页面**。
5. 本轮必须**基于当前仓库已有代码迭代**，不能另起一套页面体系。

你还特别要求：

- 方案必须结合当前项目真实情况；

## 3.4 页面先行阶段的数据策略

因为这轮先做页面，建议采用：

- **优先调用真实接口** `smartReminder`（后续接入）；
- 若接口未上线 / 返回异常，则走**本地 mock fallback**；
- 这样页面、交互、跳转可以先稳定，不会卡死在后端未完成上。

建议封装：

```js
async requestReminderWithFallback(payload) {
  try {
    const res = await wx.cloud.callFunction({
      name: 'smartReminder',
      data: payload
    })

    if (res.result?.code !== 200) {
      throw new Error(res.result?.message || 'smartReminder failed')
    }

    return normalizeReminderResult(res.result.data)
  } catch (error) {
    logWarning('daily.requestReminderWithFallback', 'use mock reminder result', {
      message: error.message
    })
    return buildMockReminderResult(payload)
  }
}
```

## 3.5 跳转试穿页的推荐上下文传递方式

本轮不要把推荐衣物完整对象塞进 URL，最稳方案是：

1. 智能提醒页生成：
   - `selectedClothesIds`
   - `title`
   - `tips`
2. 把它写入本地缓存：

```js
wx.setStorageSync('smartReminderTryonEntry', {
  source: 'daily',
  requestId,
  title,
  selectedClothesIds,
  tips,
  createdAt: Date.now()
})
```

3. 跳转到：

```js
wx.navigateTo({ url: '/pages/tryon/tryon?source=smartReminder' })
```

4. `tryon` 页 `onLoad/onShow` 再读取缓存并预选衣物。

这样可以避免：

- URL 过长；
- 图片 URL 编码膨胀；
- 页面之间对象协议难维护。

### 2.4 为什么这样做最稳

因为你要的是“顶部固定图片”，但页面底部又要输入框，直接全局 `position: fixed` 容易造成：

- 小程序键盘遮住输入框；
- 页面滚动穿透；
- 不同机型安全区错位。

所以本轮推荐：

- hero 区固定在 flex 顶部；
- content 区滚动；
- composer 区固定在底部 flex；
- 这是当前小程序里最稳的做法。

```js
// pages/daily/daily.helpers.js
function normalizeTemperature(tempText) {
  if (typeof tempText === 'number') return tempText
  if (!tempText) return null

  const matched = String(tempText).match(/-?\d+(\.\d+)?/)
  return matched ? Number(matched[0]) : null
}

function buildWeatherSuggestion({ temp, text, city }) {
  const temperature = normalizeTemperature(temp)
  const weatherText = text || '未知天气'
  const cityText = city || '当前城市'

  if (temperature === null) {
    return `${cityText}${weatherText}，建议按体感温度灵活搭配。`
  }

  if (weatherText.includes('雨')) {
    return `${cityText}${weatherText}，当前约 ${temperature}°C，建议优先选择防水外套、长裤，并准备雨具。`
  }

  if (temperature <= 10) {
    return `${cityText}${weatherText}，当前约 ${temperature}°C，建议选择保暖内搭、外套和包脚鞋。`
  }

  if (temperature >= 28) {
    return `${cityText}${weatherText}，当前约 ${temperature}°C，建议选择轻薄透气搭配并减少叠穿。`
  }

  return `${cityText}${weatherText}，当前约 ${temperature}°C，适合做通勤或日常层次搭配。`
}
```

> 实现时注意：上面代码块里 `normalizeTemperature` 需要从同一个 helper 文件中正确导出，不要自引用写错。推荐把天气相关函数集中放在一个 helper 文件里统一导出。

fetchWeatherFromAmap(lng, lat) {
  const key = this.data.amapKey

  if (key === '请在这里填入你的高德Web服务Key') {
    const fallbackWeather = { temp: '25°C', text: '晴', icon: '☀️' }
    this.setData({

      currentDateLabel: buildDateLabel(new Date()),

        ...fallbackWeather,
        city: '北京市'
      })
    })
    wx.hideNavigationBarLoading()
    return
  }

  wx.request({
    url: `https://restapi.amap.com/v3/geocode/regeo?location=${lng},${lat}&key=${key}`,
    success: (res) => {
      try {
        const adcode = res.data?.regeocode?.addressComponent?.adcode
        let city = res.data?.regeocode?.addressComponent?.city
        if (!city || !city.length) {
          city = res.data?.regeocode?.addressComponent?.province || '当前城市'
        }

        this.setData({ currentCity: city, currentDateLabel: buildDateLabel(new Date()) })
        this.requestWeatherByAdcode(adcode, city)
      } catch (error) {
        logError('daily.fetchWeatherFromAmap', error)
        wx.hideNavigationBarLoading()
      }
    },
    fail: (error) => {
      logError('daily.fetchWeatherFromAmap.request', error)
      wx.hideNavigationBarLoading()
    }
  })
}

Page({
  data: {
    smartReminderEntry: null,
    ...
  },

  onLoad(options) {
    this.pendingReminderSource = options.source || ''

  async loadRealClothes() {
    ...
    this.setData({ clothesList: cleanData })
    this.filterClothes()
    this.applySmartReminderEntryIfNeeded(cleanData)
  },

  applySmartReminderEntryIfNeeded(clothesList) {
    if (this.pendingReminderSource !== 'smartReminder') return

    try {
      const entry = wx.getStorageSync('smartReminderTryonEntry')
      if (!entry || entry.source !== 'smartReminder') {
        logWarning('tryon.applySmartReminderEntryIfNeeded', 'entry missing')
        return
      }

      const clothesMap = new Map(clothesList.map(item => [item._id, item]))
      const matched = (entry.selectedClothesIds || [])
        .map(id => clothesMap.get(id))
        .filter(Boolean)
        .map((item, index) => ({
          ...item,
          x: 30 + (index % 2) * 120,
          y: 30 + Math.floor(index / 2) * 140
        }))

      if (entry.selectedClothesIds?.length && !matched.length) {
        logWarning('tryon.applySmartReminderEntryIfNeeded', 'no matched clothes found', {
          selectedClothesIds: entry.selectedClothesIds
        })
      }

      this.setData({
        smartReminderEntry: entry,
        selectedClothes: matched
      })
    } catch (error) {
      logError('tryon.applySmartReminderEntryIfNeeded', error)
    }
  }
